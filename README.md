# SurviveFromKevin
케빈의 집에서 살아남는 3D 퍼즐 게임입니다.

* 팀명 : 박종수
* 팀원 : 송유섭, 이주연, 이승연, 최하나
* 진행 기간 : 2023.09.25 ~ 2023.10.04

<br/>
<br/>

## 데모 영상
유튜브에서 시연영상을 확인하실 수 있습니다 : [시연영상](https://youtu.be/4t9cnIQ8CX0)

## 규칙
### 1) PR 규칙
- 코드리뷰 필수

<br/>

### 2) 브랜치명 규칙
- feature/기능명/개발자명
- feature/#이슈명


<br/>

### 3) 커밋 규칙
- commit message 규칙
commit 제목:
[type] 작업한 내용 요약 (-작성자명)
commit message 내용:
[type]
수정한 파일 경로 (너무 많을 경우 알아볼 정도만)
각 파일에 대한 수정한 내용 설명
(-작성자명)
(이슈 사용할 경우 이슈번호 붙이기)
